# 📚 Library Management System

A Windows Forms-based Library Management System designed to help manage book inventories, borrowers, and lending operations in a simple and user-friendly interface.

## 🛠 Features

- 📘 Add, edit, delete book records
- 👤 Manage member/borrower information
- 🔁 Issue and return books
- 📅 Track due dates and overdue items
- 🔍 Search functionality for books and users
- 🖥 Built with Windows Forms (.NET Framework)
